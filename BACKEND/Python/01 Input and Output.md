## Printing

```python
print("hello, world")
```

- This will print a string - "hello, world"
- `print` is a function that takes an argument (what to print)

## Taking Input

```python
name = input("What's your name? ")
```

- `input` is a function that takes an argument (shows it to user)
- `input` returns back the value typed by user
- `name` is a variable here that saves the value typed by user

## Printing Dynamic Values

```python
name = input("What's your name? ")
print("Hello,"name)
print(f"Hello, {name}")
```

- Passed two arguments that prints string and a dynamic value
- Format strings `f"Hello, {name}"` will print the value of name
