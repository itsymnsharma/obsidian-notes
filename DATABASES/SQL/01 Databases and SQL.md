## What is a Database

- A collection of data organised for creating, reading, updating and deleting
- A database management system (DBMS) is a way to interact with a database using a graphical interface or textual language.
- Examples of DBMS: MySQL, Oracle, PostgreSQL, SQLite, Microsoft Access, MongoDB etc.

## What is SQL

- It stands for Structured Query Language
- A language via which you can create, read, update and delete data in a database
- It's a structured language which means it follows a pattern and not like a natural day to day language
